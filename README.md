## About  
Youtube クライアントアプリ  

### Feature  
* アカウントなしでお気に入りに登録できる  
* お気に入りに登録した動画を自動再生できる  
* 動画をループ再生できる  

### Support  
API Level 21 ( Android 5.0 )

### Build  
string リソースに YoutubeAPI のキーを追加する  

```
<string name="api_key">YOUR_YOUTUBE_API_KEY</string>
```
